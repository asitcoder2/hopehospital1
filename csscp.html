background Properties (COLOR)

1- background-color:color name

koi bhi colorname likh sakte hai

2-hexadecimal code

(0-9&A-F)16 hote hai
# (has symble)ke sath likha jata hai
ex. #00ffff,#A058 etc.

3-rgb(red green blue)

isame (0-255)tak value hoti hai 3no ki
3no ki value ko (0 0 0)karne par black color milta hai
&3no ki value ko (255,255,255) karne par white color milta hai

4-rgba(red green blue alpha)

alpha(rgba) ka use transparent color ke liye karte hai

5-hsl(hue saturation lightness)

h (hue)ki value degree me hoti hai (0-360)
s(saturation)ki value % me hoti hai (0%-100%)tak 0% krenge to gray color & 100% krenge to jo color hoga vo dark hota jayega
l(lightness)ki valuebhi % me hoti hai (0%-100%)tak 0% krengeto black & 100% krengeto white
